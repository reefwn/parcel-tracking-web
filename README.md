# Server

> don't forget to run postgres and redis services

> database connection can be configured on constants.ts

## Install packages
```
yarn install
```

## Compile to dist folder
```
tsc
```

## Run service
```
yarn dev
```

# Web

## Install packages
```
yarn install
```

## Compile
```
yarn dev
```
